== Installation ==

= Automated Installation =

1. Go to your WordPress admin dashboard, then find the “Plugins” menu.
2. Click on “Add New” and search for “LiveCaller”.
3. Activate the LiveCaller plugin through the `Plugins` menu in WordPress.
4. Click the `LiveCaller` menu on the left.
5. Copy `Widget ID` from the [registration page](https://www.livecaller.io/register?utm_source=wordpress.org&utm_medium=integration&utm_campaign=Wordpress) and insert in Livecaller's settings page.

= Manual Installation = 

1. Upload `livecaller-widget` directory to WordPress plugins directory (`/wp-content/plugins/`).
2. Activate the LiveCaller plugin through the `Plugins` menu in WordPress.
3. Click the `LiveCaller` menu on the left.
4. Copy `Widget ID` from the [registration page](https://www.livecaller.io/register?utm_source=wordpress.org&utm_medium=integration&utm_campaign=Wordpress) and insert in Livecaller's settings page.
